Full power down mode

Set macro cPWR_UsePowerDownMode and cPWR_FullPowerDownMode to 1 to enable full power down mode. The main digital domain is switched off, flash is off and the required SRAM blocks are powered up to retain contents. All peripheral clocks and all clock sources are switched off keeping 32 kHz clock running. ARM MCU is in deep sleep mode, BLE controller is in deep sleep mode and it can wake-up entire chip in case of BLE events.

Possible wake up sources: BLE core, RTC, GPIO, I2C0, SPI0, USART0, BODVBAT.

Calling PWR_ChangeDeepSleepMode(1) or PWR_ChangeDeepSleepMode(3) will set the system go to power down mode if it is in idle state.

Calling PWR_ChangeDeepSleepMode(4) will set system enter into deep power down mode.
