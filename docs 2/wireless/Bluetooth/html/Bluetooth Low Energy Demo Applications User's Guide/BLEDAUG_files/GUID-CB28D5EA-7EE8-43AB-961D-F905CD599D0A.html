K32W0/QN9090 platform

K32W0/QN9090 platform is based on the K32W0/QN9090 Bluetooth Low-Power SoCs, which incorporate an Arm Cortex-M4 core. The platform is configured to operate at various frequencies, up to 48 MHz. It has 640 KB embedded flash and 152 KB RAM. For detailed information about the platform, see the appropriate board user's guide.
