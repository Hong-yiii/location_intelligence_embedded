Usage

In the Power Profiling application, it can work with different work mode. And user can use this application to test the power consumption in different work mode.

Pressing MODESW button to switch work mode in turn. And in mode 4, QN9090 starts to advertise and can be connected. If Bluetooth LE connection is established, Pressing MODESW button, it disconnects the current link. The user may wish to plug a USB-to-Serial converter on pins 10 and 11 so as to visualize mode changes on a terminal window.
