User interface

After flashing the board, if cPWR_FullPowerDownMode is enabled, the device enters into power down mode (STLED1, STLED2 are off). To switch to the next work mode, press MODESW button.

There are 5 different work modes for the Power Profiling, and user can use MODESW button to change work mode of QN9090.

Note: If using QN9090/K32W061, see application note AN12708. This application note includes guidance on modification required to achieve the lowest power consumption.

See details below for four power mode.

Work Mode Description
Mode 0 Power down RAM Off OSC On
Mode 1 Deep power down RAM Off OSC Off
Mode 2 Bluetooth LE LL active, MCU sleep
Mode 3 Bluetooth LE LL active, MCU active
Mode 4 Power Down RAM Retention OSC On

When it enters mode 4, the power profiling application is in connectable advertising state, with a 1s advertising interval. Between advertising events, the systems goes to power-down mode, which implies RAM retention, with 36 kbytes of RAM held. See details below for hardware references.

Platform MODESW STLED1 STLED2
QN9090/K32W061 USERINTERFACE DS2 DS3
