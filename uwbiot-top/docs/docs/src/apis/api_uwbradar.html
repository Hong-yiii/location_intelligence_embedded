uint8_t UwbRadarApi_SetBreathingConfigs(uint8_t breathing_update_rate, uint8_t radar_frame_repeat_interval, UwbRadarApi_Breathing_Measurement_Callback_t *pBdCback)¶

API to set Breathing Detection configuration for Radar Lib. API takes breathing configurations and callback as input. The callback can be used to Notifies the result via callback.

Parameters

  • breathing_update_rate – [in] the interval at which the pBdCback will be called and it is a multiple of the RFRI.

  • radar_frame_repeat_interval – [in] Radar Frame Repetition Interval in milliseconds can be 100, 50, 20, 10 eg if RFRI=100 and breathing_update_rate=10 then p_cback is called every second

  • pBdCback – [in] pBdCback the callback to Notify the Results.

Return values

  • kStatus_Success – If all params are valid.

  • kStatus_Fail – In case RFRI is not one of the allowed values.

  • kStatus_Invalid – If breathing_update_rate param less than or equal to 0 and callback is NULL.

void UwbRadarApi_SendCirSample(uint8_t cir_data[], uint16_t length)¶

This API takes CIR sample as input and applies the breathing/Presence detection algorithms and provides the results via callbacks. bd_Measurement_callback_t and pd_Measurement_callback_t

Parameters

  • cir_data – [in] of Radar Notification.

  • length – [in] of the cir data.

uint8_t UwbRadarApi_SetPresence_Configs(float detection_range_min, float detection_range_max, float detection_sensitivity, uint8_t radar_frame_repeat_interval, uint8_t reaction_time, uint8_t settling_time, UwbRadarApi_Presence_Measurement_Callback_t *pPdCback)¶

API to Set Presence detection configuration for Radar Lib. This API takes Presence detection configurations and callback as a inputs and callback can be used to Notifies the result via callback.

Parameters

  • detection_range_min – [in] the minimum distance for presence detection in centimetre.

  • detection_range_max – [in] the maximum distance for presence detection in centimetre detection_range_max <= TapsToCM(MAX_CIR_TAPS), MAX_CIR_TAPS = 128. i.e TapsToCM(MAX_CIR_TAPS) ((numTaps - 6) * 15) interanl to Radar Lib.

  • detection_sensitivity – [in] threshold for peak selection.

  • radar_frame_repeat_interval – [in] Radar Frame Repetition Interval time between CIRs in ms.

  • reaction_time – [in] Reaction time in ms in which the device should react.

  • settling_time – [in] pPdCback the callback to Notify the Results.

  • pPdCback – [in] pPdCback the callback to Notify the Results.

Return values

  • kStatus_Success – if all params are valid

  • kStatus_Invalid – if all params are less than 0 and callback is Null

  • kStatus_Fail – otherwise
