Host stack configurations for QN9090 and K32W061

The application can configure and share variables with the Bluetooth Low Energy Host Stack library. A dedicated configuration structure called bleHostGlobalConfig_t is available and defined in ble_globals.c. The RAM address of this structure is provisioned in the Host Stack library by calling the function Ble_HostConfigInit. Doing this allows to configure the host stack at run time.
